  1️⃣  server.emit('event', data)
      └── Gửi đến TẤT CẢ clients đang kết nối
      
      ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐
      │    ✅   │ │    ✅   │ │    ✅   │ │    ✅   │ │    ✅   │
      └─────────┘ └─────────┘ └─────────┘ └─────────┘ └─────────┘
         A           B           C           D           E

  ─────────────────────────────────────────────────────────────────────────

  2️⃣  server.to('room-id').emit('event', data)
      └── Gửi đến TẤT CẢ clients trong room đó
      
      Room "group-123"              Room "group-456"
      ┌─────────┐ ┌─────────┐      ┌─────────┐ ┌─────────┐
      │    ✅   │ │    ✅   │      │    ❌   │ │    ❌   │
      └─────────┘ └─────────┘      └─────────┘ └─────────┘
         A           B                 D           E

  ─────────────────────────────────────────────────────────────────────────

  3️⃣  client.to('room-id').emit('event', data)
      └── Gửi đến TẤT CẢ clients trong room, NGOẠI TRỪ người gửi
      
      Room "group-123" (A là người gửi)
      ┌─────────┐ ┌─────────┐ ┌─────────┐
      │    ❌   │ │    ✅   │ │    ✅   │
      └─────────┘ └─────────┘ └─────────┘
         A           B           C
      (sender)
